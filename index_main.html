The Basics

SWAPI is an open source web application programming interface. It utilizes a set of Hypertext Transfer Protocol (HTTP) requests to return structured data responses. The responses are in JSON format by default. The API is a representational state transfer(REST) systle web resource.


To get your blasters ready to use the API, the root must be explored. The SWAPI root URL is:

						
http://swapi.co/api/

This base URL is used with the attributes and endpoints. To get the root API, you concatenate the base URL with one of the primary attributes. So to search for information on films, it would look like: 

						
http://swapi.co/api/films/

By going to this URL, all basic information on films in the database will be returned. These six attributes are used in combination with primary identification numbers for each specific data object to return detailed information.
